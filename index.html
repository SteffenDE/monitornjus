MonitorNjus

Python (WSGI, Flask) Anwendung zur Darstellung von verschiedenen Webinhalten u.a. zeitgesteuert im Browser.

View on GitHub Download .zip Download .tar.gz

Was ist MonitorNjus?

MonitorNjus ist eine Python WSGI Anwendung, die dafür gemacht worden ist, die Vertretungspläne in einer Schule anzuzeigen. Die geschieht zeitgesteuert, abwechselnd mit aktuellen Meldungen, etc.

Inhalte

Alle Inhalte, die ein Browser anzeigen kann, können im Admin-Panel als Link gespeichert werden und werden, wenn sie aktiv sind im Browser angezeigt. Dies geschieht standardmäßig im 50:50 Modus, d.h. der Anzeigefläche wird in der Hälfte geteilt und es können verschiedene Inhalte links und rechts dargestellt werden.
